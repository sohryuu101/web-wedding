# Wedding Invitation Website

This is a static demo of the wedding invitation website.

## Live Demo

Visit the demo at: [https://sohryuu101.github.io/web-wedding/demo](https://sohryuu101.github.io/web-wedding/demo)

## Features

- Beautiful wedding invitation templates
- Modern Elegance theme
- RSVP functionality (demo mode)
- Photo galleries
- Event details and countdown

## Note

This is a static version hosted on GitHub Pages. For the full application with backend features, deploy the project with a server backend.

The home page automatically redirects to the demo when accessed on GitHub Pages.
